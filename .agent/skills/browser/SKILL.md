# Browser Skill
This skill provides best practices for using browser tools to research and validate UI components.

## Instructions
- Use `search_web` for gathering documentation on libraries and APIs.
- Use `browser_subagent` for interactive testing of the application's UI.
- When researching, prioritize official documentation and reputable sources.
- Ensure that the browser environment is properly initialized before running complex tasks.
